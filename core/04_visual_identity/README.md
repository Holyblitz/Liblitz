# Direction artistique de Liblitz

L'identité visuelle de Liblitz ne cherche pas à impressionner.

Elle cherche à accueillir.

L'objectif n'est pas de créer une interface futuriste, spectaculaire ou démonstrative.

L'objectif est de créer un lieu dans lequel on a naturellement envie de revenir.

Lorsqu'une personne ouvre Liblitz, elle ne doit pas avoir l'impression d'ouvrir un logiciel.

Elle doit avoir la sensation de rentrer dans un endroit familier, calme et chaleureux.

Liblitz représente un refuge intellectuel.

Un espace où l'on réfléchit, où l'on construit, où l'on apprend et où l'on prend le temps.

Son univers est inspiré d'une cabane en bois au cœur d'une forêt de pins.

La technologie est présente.

Mais elle reste discrète.

Elle s'efface derrière l'expérience.

Chaque élément graphique doit renforcer cette sensation de calme :

- des matériaux naturels ;

- une lumière chaude ;

- des couleurs douces ;

- une bibliothèque remplie de livres ;

- une tasse de café ou de thé ;

- une fenêtre ouverte sur la forêt ;

- un bureau vivant où naissent les projets.

Liblitz ne vit pas dans un laboratoire.

Elle vit dans un atelier.

L'interface n'est pas pensée comme un tableau de bord.

Elle est pensée comme une pièce dans laquelle on entre.

Cette philosophie devra guider toutes les décisions artistiques futures.
