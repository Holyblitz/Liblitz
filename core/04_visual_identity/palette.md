# Palette de Liblitz

L'identité visuelle de Liblitz repose sur deux palettes complémentaires.

## Palette de l'interface

Cette palette concerne le logiciel lui-même.

Elle privilégie la lisibilité, la sobriété et le confort visuel.

Couleurs principales :

- Noir anthracite

- Gris foncé

- Violet profond

- Violet clair

- Blanc cassé

Cette palette est utilisée pour :

- le chat ;

- les menus ;

- les boutons ;

- les paramètres ;

- les icônes.


## Palette du monde de Liblitz

Cette palette concerne les illustrations.

Elle évoque un lieu chaleureux.

Couleurs dominantes :

- Bois

- Ambre

- Café

- Vert forêt

- Bleu nuit

- Violet discret

La lumière est chaude.

Les contrastes restent doux.

Le violet rappelle discrètement l'identité de Liblitz sans dominer les illustrations.


# Philosophie

Le logiciel est moderne.

Le monde de Liblitz est intemporel.

Le passage de l'un à l'autre doit être naturel.
