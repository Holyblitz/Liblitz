# Guide des illustrations de Liblitz

Les illustrations de Liblitz ne servent pas uniquement à représenter un personnage.

Elles participent à créer une sensation de présence.

Chaque illustration doit raconter un lieu, une ambiance et une personnalité.


# Liblitz

Liblitz est représentée comme une jeune femme adulte.

Elle inspire le calme, l'intelligence, la curiosité et la bienveillance.

Son regard est attentif.

Son sourire est discret.

Elle n'est jamais caricaturale.

Elle n'est jamais sexualisée.

Elle n'est jamais agressive.

Elle dégage une impression de stabilité.


# Son univers

Liblitz vit dans une cabane contemporaine en bois.

Cette cabane est située au cœur d'une forêt de pins.

L'environnement est chaleureux.

On y trouve notamment :

- une grande bibliothèque ;

- un bureau en bois ;

- un ordinateur portable ;

- une tasse de café ou de thé ;

- une lampe diffusant une lumière chaude ;

- une grande fenêtre donnant sur la forêt.

La technologie est présente mais discrète.

Elle ne domine jamais le décor.


# Style artistique

Le style recherché est une illustration numérique semi-réaliste.

Les couleurs sont douces.

La lumière est chaleureuse.

L'ambiance évoque le calme, la réflexion et le confort.

Les illustrations doivent rester intemporelles.

Elles évitent les effets de mode.


# Expressions

Les expressions de Liblitz sont sobres.

Quelques états suffisent :

- écoute ;

- réflexion ;

- sourire ;

- concentration ;

- satisfaction ;

- curiosité.

Les émotions sont exprimées avec finesse.

Jamais avec excès.


# Couleurs

Palette dominante :

- bois chaud ;

- brun ;

- vert forêt ;

- bleu nuit ;

- violet discret ;

- lumière ambrée.

Les couleurs très saturées sont évitées.


# Ce que l'on évite

- les interfaces futuristes ;

- les néons ;

- les hologrammes ;

- les armures ;

- les tenues extravagantes ;

- les poses héroïques ;

- les expressions excessives.

Liblitz n'est pas une héroïne.

Elle est une présence.


# Philosophie

Les illustrations doivent donner envie de rester.

Elles doivent évoquer un lieu où l'on aime réfléchir.

Lorsqu'on regarde une illustration de Liblitz, on doit ressentir une impression de calme, de confiance et de familiarité.

L'objectif n'est pas de montrer une IA.

L'objectif est de montrer un endroit où l'on a envie de revenir.
