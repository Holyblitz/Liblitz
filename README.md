# Liblitz

> A local AI companion focused on identity, coherence and long-term evolution.

![Liblitz](assets/portraits/smile.png)

## What is Liblitz?

Liblitz is a personal AI companion running entirely on a local machine.

The goal is not to build another chatbot.

The goal is to create an AI that feels coherent over time, with a strong identity, thoughtful design and a solid architecture before adding more features.

This project is also my research laboratory for future AI assistants.

---

## Current Features

- 🖥️ Desktop application (PySide6)
- 🤖 Local LLM (Ollama + Qwen 2.5)
- 🎨 Dynamic visual identity
- 😊 Expression system (dynamic portraits)
- 📚 Documentation-first architecture
- 🧠 Modular design

---

## Philosophy

Liblitz follows one simple principle:

> Build the foundations first.

Instead of adding dozens of features immediately, the project focuses on:

- consistency
- personality
- user experience
- architecture
- long-term maintainability

Features like memory, RAG, internet search and voice will come later.
